# プロジェクト要件定義書：喋Lang (シャベラン)

## 1. プロジェクト概要
### 何を作るか
外国語学習者が、日常生活で出会った知らない単語やフレーズを音声、文字、画像で瞬時に記録し、効率的に復習できる多機能メモアプリ。

### 解決したい課題
* 現在の外国語学習におけるメモは、音声や画像を一緒に記録するのが面倒。
* 後で関連する情報を探し出すのが困難。
* 発音をすぐに確認できない不便さがある。

### 解決アプローチ
多様な形式（音声、文字、画像）で直感的にメモできる機能と、記録した情報を効率的に整理・復習できる仕組みを提供。学習意欲の維持とコミュニケーション能力向上を支援する。

## 3. アプリの価値
* **主な価値:** 「すぐに記録できる」という安心感の提供と、記憶定着の効率化。
* **既存手段との違い:** 音声・文字・画像を一つのメモに統合でき、フラッシュカード形式の復習まで一貫して行える点。

---

## 4. 利用シナリオ
### シナリオ1：カフェで新しい単語を記録する
1.  **起動:** メモ一覧画面から「メモ作成ボタン」をタップし、素早く作成画面を開く。
2.  **音声録音:** 聞き取った単語の発音を音声メモとして録音。
3.  **情報入力:** 調べた意味、品詞、例文を文字で入力。
4.  **保存・分類:** 保存ボタンをタップ。既存カテゴリーをプルダウンから選択、または横の「＋」ボタンで新規作成して分類。

### シナリオ2：海外旅行中に看板の単語を記録する
1.  **撮影:** カメラ機能で看板等の画像メモを撮影（画像はアイコン表示、文字認識は現時点では対象外）。
2.  **追記:** 意味や自分なりの例文を文字で入力。
3.  **音声追加:** 必要に応じて自分の発音を録音。
4.  **保存:** カテゴリーに分類して保存。

### シナリオ3：記録した単語を効率的に復習する
1.  **開始:** 復習機能を選択。単語帳形式で単語が表示される。
2.  **ヒント:** 必要に応じて意味や画像を表示。
3.  **発音評価:** ユーザーが発音。**Azure Pronunciation Assessment**により判定。
4.  **判定結果:** * **クリア:** 正確な場合。
    * **再挑戦:** 不正確な場合、単語帳の最後尾へ。間違えた音素も表示される。
5.  **操作:** スキップも可能。発音するかスキップで次の単語へ。

## 5. UI/UX 仕様詳細
* **最初の画面:** メモ一覧画面を表示。
* **メモ一覧の表示形式:**
    * カテゴリー別、日付順で表示。
    * 「単語、意味、画像（アイコン）、登録年月日」をカード形式でリスト表示。
    * カードタップで詳細表示・編集が可能。
* **カテゴリー管理:** メモ作成画面のプルダウン横に、新規作成用の「＋」ボタンを配置。

---

## 6. 技術要件・外部連携
* **音声評価:** Azure Pronunciation Assessment (Microsoft)
* **画像管理:** 撮影した画像をメモのアイコンとして保持。
